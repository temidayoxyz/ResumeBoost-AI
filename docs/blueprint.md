# **App Name**: ResumeBoost AI

## Core Features:

- Resume Input: Allows users to paste their resume text into a clean, user-friendly input area.
- AI Feedback Generation: Generates instant, actionable feedback on the resume using the OpenAI API to assess and provide suggestions.
- Skill Improvement Suggestions: Provides specific, targeted suggestions for improving skills based on the AI analysis of the resume.
- Readability Scoring: Calculates and displays a readability score for the resume, offering insight into clarity.
- Copy-to-Clipboard Functionality: Allows users to copy the improved resume text directly to their clipboard for easy use.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to convey professionalism and intelligence.
- Background color: Very light Indigo (#E8EAF6).
- Accent color: Soft Violet (#9575CD) for interactive elements and highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif font for a modern, machined look.
- Modern card-based layout for organized display of feedback and suggestions.
- Subtle animations for loading and feedback display to enhance user experience.
- Use professional and clean icons for resume sections and action buttons.
